SCENE 5 — INTERNAL FORM STEPS (show wc-internal-form-steps.png)
============================================================
Teach ONLY these Internal steps, one beat each:

STEP A — Identity
  Fields: Tenant name + Owner
  Tip: Name it so your team recognizes the site.

STEP B — Access scope
  Region = REQUIRED (glow)
  Sub-Region / Client / Security Group = optional
  Diagram: Region → Sub-Region → Client → Security Group

STEP C — Site URL
  Paste full SharePoint link
  Magic extract: Host + Site + Library (+ optional Folder)
  Example shape: https://contoso.sharepoint.com/sites/MyTeamSite
  No Entra secret screens.

STEP D — Test connection (Internal pipeline only)
  Chevrons: Resolve Site → Read Libraries
  Success chips: Host | Site title | Libraries
  Green badge: Connection verified
  If fail: fix URL/permissions, re-test (or inactive — brief)

STEP E — Review + Register
  Confirm tiles: Tenant, Owner, Internal, Site, Host, Library, Region
  CTA: Register connector → success toast → Browse workspace now
