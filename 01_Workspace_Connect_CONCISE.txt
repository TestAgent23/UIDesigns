WHAT IS WORKSPACE CONNECT
Workspace Connect is the module in TP Data Ingestion where users register
Internal SharePoint sites as connector profiles (tenants), verify access, and
browse libraries and files for downstream ingestion.

Path taught here = INTERNAL (platform credentials).

================================================================================
END-TO-END FLOW (memorize this diagram)
================================================================================
Home
  → Register Tenants (or open SharePoint connectors)
  → Select Internal
  → Enter Tenant name + Owner
  → Set Access scope (Region required)
  → Paste SharePoint URL (auto-extract Host / Site / Library)
  → Test connection (Resolve Site → Read Libraries)
  → Register connector
  → Use tenant
  → Connect in SharePoint Workspace
  → Browse folders / files → ingest / streaming API

REGISTER INTERNAL SHAREPOINT (STEPS)
================================================================================
1) TYPE
   Select Internal.
   Meaning: platform-managed SharePoint using platform credentials.

2) IDENTITY
   • Tenant name — clear display name (e.g. Contoso Marketing SharePoint)
   • Owner — person responsible for the connector

3) ACCESS CONFIGURATION
   • Region — REQUIRED
   • Sub-Region — optional (default All)
   • Client — optional (default All)
   • Security Group — optional search (min 5 characters)

4) SITE
   Paste full SharePoint URL, example:
   https://contoso.sharepoint.com/sites/MyTeamSite
   System extracts: Host, Site path, Library, optional Folder.
   Review extracted details before continuing.

5) VERIFY
   Click Test connection.
   Checks: Resolve SharePoint site → Read document libraries.
   Success = Connection verified (Host, Site title, Libraries).
   Fail = fix URL/permissions, Re-test; or Register as inactive.

6) REVIEW + SUBMIT
   Confirm: Tenant, Owner, Type=Internal, Site, Host, Library, Region.
   Click Register connector.
   Success → option Browse workspace now.

================================================================================
MANAGE TENANTS
================================================================================
SharePoint connectors page shows registered tenants.
Actions: Edit | Use | Delete
Views: Cards / List
Filters: search, type, Draft, access scope
Use = open that tenant in SharePoint Workspace

================================================================================
SHAREPOINT WORKSPACE (INTERNAL)
================================================================================
1) Mode: Internal
2) Select Tenant → Select Site
3) Click Connect
4) Choose document library
5) Browse folders (list or grid)
6) Preview / download files
7) Optional: Generate cURL for streaming API
   Step A: Get access token (registered app ID + API secret)
   Step B: Browse folder (optional)
   Step C: Download/stream file with filePath JSON body
